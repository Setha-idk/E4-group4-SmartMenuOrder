# project title:
smart menu order

# introduction

This project is an application designed to make food ordering from a
restaurants faster and easier. customers can browse the digital menu on
their phone, serach the menu, add items to their cart, and place orders
directly. The system also helps customer orders more efficiently. This
project provides a convenient modern solution for both customers and
restaurant staff.

# problem
• Traditional menu are hard to update. required to reprint the menu
• customer take a long time looking for items in Traditional menu, cant be search for like in app
• traditional menu can only be found/read at the restaurant 

# Solution
- help costumers order items faster
-  help restaurant by needing less waiters
- easy to pay with adding items to a cart
- user can add an order as a favourite for easier ordering in the future
- when the user order, the chiefs can see the order in the admin dashboard and can print the receipt

#  Feature List
- User registration & login
- CRUD Category 
- Browse restaurant and their menu (with images)
- Add items to cart 
- Place and track orders
- Payment integration
- Order history
- Admin panel for menu & order management
